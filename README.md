# 搬木民宿庄园 - 前端项目

## 项目简介
基于 Vue 3 + Vite + Element Plus 构建的旅游服务平台前端。
包含前台展示系统（首页、详情、预定、个人中心）和后台管理系统（用户管理、景点管理、订单管理）。

## 技术栈
- **核心框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **HTTP 请求**: Axios

## 目录结构
- `src/views`: 页面文件
- `src/layout`: 布局组件 (AdminLayout, FrontLayout)
- `src/components`: 公共组件
- `src/api`: 接口封装
